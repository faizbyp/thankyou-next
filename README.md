# Thank You, Next

A repo for learning Next JS. [Tutorial from Web Dev Simplified](https://youtu.be/NgayZAuTgwM)

## Setup

Creating Next app
```bash
npx create-next-app@latest
```

Installing Prisma
```bash
npm i prisma -D
```

Init Prisma with SQLite
```bash
npx prisma init --datasource-provider sqlite
```
